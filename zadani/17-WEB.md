
# 17 — WEB (Draft v0.1)

Status: Draft

## Účel

Webový klient dostupný z běžného prohlížeče bez instalace.

## Cíle

- Bezpečné přihlášení
- Plná funkčnost chatu
- End-to-end šifrování v prohlížeči
- Responzivní rozhraní

## Technologie

- React + TypeScript
- Web Crypto API
- Firebase SDK

## Bezpečnost

- Povinné HTTPS
- CSP
- HSTS
- Ochrana proti XSS
- Ochrana proti CSRF tam, kde je relevantní

## Budoucí kapitoly

- Více oken
- Offline režim
- Desktop instalace
- Správa zařízení
