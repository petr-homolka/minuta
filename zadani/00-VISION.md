# MINUTA — Vision

> Working draft v0.1

## Cíl projektu

Minuta je privacy-first komunikační platforma navržená tak, aby provozovatel neměl technickou možnost číst obsah komunikace.

## Základní principy

- End-to-end šifrování.
- Zero-knowledge architektura.
- Metadata jsou minimalizována.
- Obsah zpráv se po uplynutí definované doby stává neobnovitelným.
- Architektura je navržena pro škálování od desítek po miliony uživatelů.
